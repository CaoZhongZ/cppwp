27.10.15.10 Create symlink [fs.op.create_symlink]

void create_symlink(const path& to, const path& new_symlink); void create_symlink(const path& to, const path& new_symlink, error_code& ec) noexcept;

1

Effects: Establishes the postcondition, as if by POSIX symlink().

2

Postconditions: new_symlink resolves to a symbolic link file that contains an unspecified representation of to.

3

Throws: As specified in Error reporting ([fs.err.report]).

4
