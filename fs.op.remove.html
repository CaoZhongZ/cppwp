27.10.15.30 Remove [fs.op.remove]

bool remove(const path& p); bool remove(const path& p, error_code& ec) noexcept;

1

Effects: If exists(symlink_status(p, ec)), the file p is removed as if by POSIX remove(). [ Note: A symbolic link is itself removed, rather than the file it resolves to.  â€” end note ]

2

Postcondition: !exists(symlink_status(p)).

3

Returns: false if p did not exist, otherwise true. The signature with argument ec returns false if an error occurs.

4

Throws: As specified in Error reporting ([fs.err.report]).
