C.5.2.2 Type wchar_­t [diff.wchar.t]

1
#

The type wchar_­t is a distinct type rather than a typedef to an existing integral type. The token wchar_­t is a keyword in this International Standard ([lex.key]). It does not appear as a type name defined in any of <cstddef> ([cstddef.syn]), <cstdlib> ([cstdlib.syn]), or <cwchar> ([cwchar.syn]).
