16.2 Requirements [buffer.reqmts]

16.2.1 Mutable buffer sequence requirements [buffer.reqmts.mutablebuffersequence]

1

A mutable buffer sequence represents a set of memory regions that may be used to receive the output of an operation, such as the receive operation of a socket.

2

A type X meets the MutableBufferSequence requirements if it satisfies the requirements of Destructible (C++Std §[destructible]) and CopyConstructible (C++Std §[copyconstructible]), as well as the additional requirements listed in Table [tab:buffer.reqmts.mutablebuffersequence.requirements].

3

In Table [tab:buffer.reqmts.mutablebuffersequence.requirements], x denotes a value of type X or const X, and u denotes an identifier.
