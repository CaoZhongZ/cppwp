10 Declarations [dcl.dcl]

10.4 The asm declaration [dcl.asm]

1
#

An asm declaration has the form

asm-definition:
	attribute-specifier-seqopt asm ( string-literal ) ;

The asm declaration is conditionally-supported; its meaning is implementation-defined. The optional attribute-specifier-seq in an asm-definition appertains to the asm declaration. [ Note: Typically it is used to pass information through the implementation to an assembler.  â€” end note ]
