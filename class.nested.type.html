#

Type names obey exactly the same scope rules as other names. In particular, type names defined within a class definition cannot be used outside their class without qualification. [ Example:

struct X {
  typedef int I;
  class Y { /* ... */ };
  I a;
};

Y c;                            // error
X::Y d;                         // OK
X::I e;                         // OK
