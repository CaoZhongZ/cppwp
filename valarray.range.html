24 Numerics library [numerics]

24.8 Numeric arrays [numarray]

24.8.10 valarray range access [valarray.range]

1
#
In the begin and end function templates that follow, unspecified1 is a type that meets the requirements of a mutable random access iterator and of a contiguous iterator whose value_­type is the template parameter T and whose reference type is T&.
unspecified2 is a type that meets the requirements of a constant random access iterator and of a contiguous iterator whose value_­type is the template parameter T and whose reference type is const T&.
2
#
The iterators returned by begin and end for an array are guaranteed to be valid until the member function resize(size_­t, T) is called for that array or until the lifetime of that array ends, whichever happens first.
