C.5 C standard library [diff.library]

C.5.1 Modifications to headers [diff.mods.to.headers]

1
#

For compatibility with the C standard library, the C++ standard library provides the C headers enumerated in [depr.c.headers], but their use is deprecated in C++.

2
#

There are no C++ headers for the C headers <stdatomic.h>, <stdnoreturn.h>, and <threads.h>, nor are the C headers themselves part of C++.

3
#

The headers <ccomplex> and <ctgmath>, as well as their corresponding C headers <complex.h> and <tgmath.h>, do not contain any of the content from the C standard library and instead merely include other headers from the C++ standard library.

4
#

The headers <ciso646>, <cstdalign>, and <cstdbool> are meaningless in C++. Use of the C++ headers <ccomplex>, <cstdalign>, <cstdbool>, and <ctgmath> is deprecated ([depr.c.headers]).
