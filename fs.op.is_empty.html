27.10.15.19 Is empty [fs.op.is_empty]

bool is_empty(const path& p); bool is_empty(const path& p, error_code& ec) noexcept;

1

Let s be the file_status, determined as if by status(p, ec).

2

Effects: Determines s.

3

Returns: 

is_directory(s) ? directory_iterator(p) == directory_iterator() : file_size(p) == 0;

The signature with argument ec returns false if an error occurs.

4

Throws: As specified in Error reporting ([fs.err.report]).
