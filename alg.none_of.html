28 Algorithms library [algorithms]

28.5 Non-modifying sequence operations [alg.nonmodifying]

28.5.3 None of [alg.none_of]

template <class InputIterator, class Predicate> bool none_of(InputIterator first, InputIterator last, Predicate pred); template <class ExecutionPolicy, class InputIterator, class Predicate> bool none_of(ExecutionPolicy&& exec, InputIterator first, InputIterator last, Predicate pred);
1
#

Returns: true if [first, last) is empty or if pred(*i) is false for every iterator i in the range [first, last), and false otherwise.

2
#

Complexity: At most last - first applications of the predicate.
