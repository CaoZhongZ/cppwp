The BoundedRange concept specifies requirements of a Range type for which begin and end return objects of the same type. [ Note: The standard containers ( ISO/IEC 14882:2014 §[containers]) satisfy BoundedRange. â€” end note ]

template <class T>
concept bool BoundedRange() {
  return Range<T>() && Same<iterator_t<T>, sentinel_t<T>>();
}
