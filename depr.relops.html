Annex D (normative) Compatibility features [depr]

D.7 Relational operators [depr.relops]

1
#
The header <utility> has the following additions:
namespace std::rel_ops {
  template<class T> bool operator!=(const T&, const T&);
  template<class T> bool operator> (const T&, const T&);
  template<class T> bool operator<=(const T&, const T&);
  template<class T> bool operator>=(const T&, const T&);
}

#
